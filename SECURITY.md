# Security Policy

## Supported Versions

Security fixes are provided for the latest release on the `main` branch.

| Version | Supported |
| --- | --- |
| latest | yes |
| older releases | no |

## Reporting A Vulnerability

Please do not report security issues in public GitHub issues.

Preferred channel:

- Use GitHub's private vulnerability reporting (Security Advisories) for this repository.

Fallback channel:

- Open a minimal public issue requesting a private contact path if private reporting is unavailable.

Please include:

- Affected version/commit
- Reproduction steps or proof of concept
- Impact and suggested remediation (if known)

We will acknowledge receipt as soon as possible and work on remediation and coordinated disclosure.
